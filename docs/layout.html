Layout

Layout is controlled by settings object MatxLayoutSettings inside app/MatxLayout/settings.js

MatxLayoutSettings

activeLayout: Main layout. (e.g. layout1, layout2)

activeTheme: Main theme color. All valid theme colors are available inside app/MatxLayout/MatxTheme/themeColors.js. New theme color can be defined inside themeColors.js

layout1Settings: You will only modify this if you are using layout1 as activeLayout (variable location: app/MatxLayout/Layout1/Layout1Settings.js)

layout2Settings: You will only modify this if you are using layout2 as activeLayout (variable location: app/MatxLayout/Layout2/Layout2Settings.js)

footer: Options for footer display, position, & theme
